# 🎤 AI Interview Coach

🚀 Live Demo: https://ai-interview-simulator-5x3ghdyngmkefwceq9n6dy.streamlit.app/

An AI-powered interview practice web application built using Streamlit.  
It helps users prepare for interviews by generating domain-based questions, evaluating answers, and providing real-time feedback.

---

## 💡 Features

- 🎯 Multi-question interview system  
- 🧠 Domain selection (Python / AI-ML / HR)  
- 📊 AI-style answer evaluation system  
- 🔥 Real-time scoring and feedback  
- 📝 Final performance report  
- 📜 Answer history tracking  
- 🔄 Restart interview option  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🎈  
- Rule-based NLP scoring logic

## 📸 How It Works

1. Select Interview Type (Python / AI-ML / HR)  
2. Click **Start Interview**  
3. Answer generated questions  
4. Get instant scoring + feedback  
5. View final performance summary
